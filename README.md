* **TV Optimized:** Native support for Tizen Remote interactions (Keycodes: Enter, Up, Back).

* **Dynamic Physics:** Custom gravity and velocity logic for smooth "floaty" flight.
* **Parallax Environment:** Multi-layered background system (Sky, Mountains, Ground) moving at different speeds.

* **Responsive UI:** HD (1920x1080) Layout with scale-to-fit assets.
